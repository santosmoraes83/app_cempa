# Care Tech App 🏥

Projeto desenvolvido como parte da **Etapa III** do curso técnico em TI para Internet. O **Care Tech** é um sistema de gestão de pacientes para clínicas, focado em praticidade e acessibilidade.

## 🚀 Funcionalidades (CRUD Completo)
O aplicativo cumpre os seguintes requisitos funcionais:
- **[RF01] Novo Cadastro:** Registro de novos pacientes com validação.
- **[RF02] Consulta:** Listagem em tempo real de pacientes cadastrados.
- **[RF03] Busca:** Filtro de pesquisa por nome na lista de pacientes.
- **[RF04] Edição:** Alteração de dados de pacientes já existentes.
- **[RF05] Exclusão:** Remoção de registros da lista com feedback ao usuário.

## 🛠️ Tecnologias Utilizadas
- **Flutter & Dart:** Desenvolvimento multiplataforma.
- **Material Design:** Padronização visual e acessibilidade.
- **Git & GitHub:** Controle de versão e colaboração.

## 📱 Como rodar o projeto
1. Certifique-se de ter o Flutter instalado.
2. Clone o repositório: `git clone https://github.com/santosmoraes83/app_cempa.git`
3. Execute `flutter pub get` para instalar as dependências.
4. Rode o projeto com `flutter run`.
